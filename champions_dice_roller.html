💡 Usage Tips and Best Practices

🎲 Dice Rolling Tips

  • Use the target number field for all skill checks and attack rolls
  • Apply modifiers for difficulty, range, or circumstances
  • The success chance updates automatically to help you understand odds
  • Roll multiple dice for getting averages or handling multiple attacks

⚔️ Combat Usage

  • Use the damage calculator for all attack damage
  • Remember to select the correct damage type (Normal/Killing/Mental)
  • Apply appropriate defenses (PD for physical, ED for energy)
  • Track initiative with the combat tracker

📊 Character Tracking

  • Update your character's max values when they change
  • Use the recovery buttons for post-turn healing
  • Save character state before dangerous encounters
  • Reset to full health between adventures

⚡ Power Building

  • Start with base power cost from the Champions rules
  • Select multiple advantages/limitations using Ctrl+Click
  • The calculator shows both Active and Real point costs
  • Use this for quick "what if" power modifications
